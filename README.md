# Посчитайте клики по ссылкам

**Удобный инструмент для сокращения ссылок и просмотра статистики кликов** через сервис **clc.li**.

## Описание проекта
- Сокращение длинных ссылок
- Проверка количества кликов по уже сокращённым ссылкам


## Как установить

1. Убедитесь, что у вас установлен [Python 3.14.6](https://www.python.org/downloads/release/python-3146/).
2. Склонируйте репозиторий:

```bash
git clone https://github.com/igor-sos/Count-the-clicks-on-the-links.git
```
```bash
cd Count-the-clicks-on-the-links
```

### Настройка переменных окружения

Создай в корне проекта файл `.env` и добавь туда такие переменные:

```env
TOKEN=your_token_here
```

### Установите зависимости:

```Bash
pip install -r requirements.txt
```

### Как запустить:

```Bash 
python main.py
```

### Цель проекта
Код написан в образовательных целях на онлайн-курсе для веб-разработчиков dvmn.org.
